# Modern Personal Portfolio - Aditya Pawar

A premium, production-ready personal portfolio website built with React, Tailwind CSS, and Framer Motion. Featuring a smooth UI/UX, dark/light mode, and a functional contact form with Node.js/MongoDB backend.

## 🚀 Features

- **Premium Design:** Modern glassmorphism UI with a professional color palette.
- **Responsive:** Fully optimized for mobile, tablet, and desktop.
- **Animations:** High-end scroll-reveal and entrance animations using Framer Motion.
- **Dark Mode:** Seamless one-click dark/light mode toggle.
- **Projects:** Dynamic project gallery with categorization and detailed modals.
- **Contact Form:** Integrated backend for message storage and email notifications.
- **SEO Ready:** Optimized meta tags and semantic HTML.

## 🛠️ Tech Stack

**Frontend:**
- React.js (Vite)
- Tailwind CSS
- Framer Motion (Animations)
- React Icons & React Scroll

**Backend:**
- Node.js & Express
- MongoDB (Mongoose)
- Nodemailer (Email service)

## 📦 Setup Instructions

### 1. Frontend Setup
```bash
# Navigate to root
npm install
npm run dev
```

### 2. Backend Setup
```bash
cd backend
npm install
# Create .env file based on .env.example
npm run dev
```
